# Codveda Java Development Internship

This repository contains the projects completed during my **Java Development Internship at Codveda**.

The tasks are organized into three levels based on complexity.

# Level 1 Projects

## 1. Basic Calculator
A Java console application that performs basic arithmetic operations.

Features:
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

Concepts used:
- Methods
- Switch statements
- User input using Scanner

## 2. Number Guessing Game
A game where the user guesses a randomly generated number between **1 and 100**.

Features:
- Random number generation
- Hints for too high or too low
- Loop until correct guess

Concepts used:
- Loops
- Conditional statements
- Random class
